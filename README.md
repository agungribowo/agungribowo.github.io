# agungribowo.github.io

Personal website and blog for **agungribowo**.

- **Maintainer:** agungribowo (https://github.com/agungribowo)

This repository contains a Vite + React + TypeScript site. I adapted and significantly modified the original starter that was imported earlier; the site is now maintained by me.

## About

This project is a personal site built with Vite, React, and TypeScript. It hosts static pages, blog posts, and assets.

## Development

Install dependencies and run the dev server:

```bash
pnpm install
pnpm dev
```

Build for production:

```bash
pnpm build
```
